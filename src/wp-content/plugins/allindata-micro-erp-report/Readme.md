# Micro ERP Report

The Report module of the All.In Data Micro ERP environment aggregates and renders key fact data.

## Dependencies

- Micro ERP Core >= 1.0
- Elementor Page Builder >= 2.5
- Micro ERP Auth >= 1.0
- Micro ERP MDM >= 1.0
- Micro ERP Planning >= 1.0
- Micro ERP Resource >= 1.0
- Micro ERP Subordination
